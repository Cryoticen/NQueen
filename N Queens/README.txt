To execute the program, follow these instructions:

- Open the Command Prompt
- Type: "cd pathName" (pathName = the complete path of the src folder)
- Type: "javac Main.java Chromosome.java GeneticAlgorithm.java"
- Type: "java Main"
- Follow the prompt to enter data
	- Enter 0 for running the program with default values:
	  	Number of Queens: 8
	  	Population Size: 1000
		Maximum Steps: 1000
	- Enter any number of queens and for the rest of the required fields
